Facial Emotion Recognition using 2 stage learning
Computer Vision
Machine Learning
Image Processing

'Develop a pipeline that uses machine learning to identify emotions in an image of a face' This work developed a pipeline for recognition of emotion based on an image of the face. This was done in two stages. 1. Features were extracted from the face using a bank of Gabor Filters at different spatial frequencies and orientations. These features were used to train Facial Action Unit(FAU) Classifiers that identify the presence of micro-expressions in the face. These classifiers were Support Vector Machines(SVMs) used to produce probabilities representing the intensities with which the FAUs are present in the face. 2. A decision tree was trained to make use of these scores to classify the emotion seen in the face. This was pursued as part of a course project for the Machine Learning course.
